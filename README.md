# MiniMerge

MiniMerge is a small browser app for combining pages from PDFs and images into one new PDF.

## What it does

- Drop in PDF, JPG, or PNG files.
- See every page as a thumbnail.
- Reorder pages with drag and drop.
- Select single, multiple, range, or all pages.
- Remove selected pages.
- Export the current order as one merged PDF.

## Shortcuts

- `Shift` + click: select a range
- `Cmd`/`Ctrl` + click: toggle a page
- `Cmd`/`Ctrl` + `A`: select all pages
- `Backspace` or `Delete`: remove selected pages

## Run locally

```bash
npm install
npm run dev
```
